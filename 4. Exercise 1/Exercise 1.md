# 🧪 Exercise 1: Using `print()` Function

---

## 🎯 Objective

Practice using the `print()` function to display different types of output.

---

## 📌 Questions

### 1. Print your name

Write a Python program to display your name.


---

### 2. Print your age

Display your age using the `print()` function.



---

### 3. Print a sentence

Print the following sentence:
"My name is ___ and I am learning Python."



---

### 4. Print multiple values

Print your name and age in one line.



---

### 5. Print a calculation

Display the result of:
5 + 10



---

## ✅ Sample Solutions

```python
print("John")
print(20)
print("My name is John and I am learning Python.")
print("John", 20)
print(5 + 10)
```
