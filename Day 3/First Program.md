# 🐍 First Python Program using `print()`

---

## 📌 Introduction

The first program that most beginners write in Python is the **"Hello, World!"** program.
It is used to understand the basic syntax of Python and how output works.

---

## 🧠 What is `print()`?

`print()` is a **built-in function** in Python used to display output on the screen.

---

## 💻 Syntax

```python
print("Your message here")
```

---

## 📝 Example: Hello World Program

```python
print("Hello, World!")
```

---

## ▶️ Output

```
Hello, World!
```

---

## 🔍 Explanation

* `print()` → Function used to display output
* `"Hello, World!"` → String (text inside quotes)
* Python executes this line and shows the text on the screen

---

## 🧪 More Examples

```python
print("Welcome to Python")
print(10)
print("Sum:", 5 + 3)
```

---

## ▶️ Output

```
Welcome to Python
10
Sum: 8
```

---

## ⚡ Key Points

* Always use quotes (`" "` or `' '`) for text
* `print()` can display text, numbers, and expressions
* It is one of the most commonly used functions in Python

---

## 🚀 Conclusion

The `print()` function is the first step in learning Python.
It helps you understand how Python communicates with the user.

---

⭐ *Practice printing different messages to get comfortable with Python!*
